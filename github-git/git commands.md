## Allow empty commit
```
git commit --allow-empty -m "message"
```
## Checkout branch
```
git checkout branch
```
## Merge branch
```
git merge branch
```
## Add new files
```
git add .
```

## Switch branch
```
git switch branch
```

## Restore changes
```
git restore .
```

## Checkout and create new branch
```
git checkout -b new-branch-name
```
